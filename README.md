# PulseMetrics

Modern SaaS analytics dashboard with marketing landing page and fully functional admin panel.

## Features

### Landing Page
- Modern gradient design with glassmorphism
- Dark/Light theme toggle
- Pricing table
- Responsive layout

### Dashboard
- **Overview** — Key metrics, revenue charts, channel analytics
- **Analytics** — Session trends, performance radar, traffic sources
- **Customers** — Table with search, sort, filter, pagination
- **Reports** — Report management by type
- **Messages** — Chat interface
- **Settings** — Profile, Security, Notifications, Billing

## Tech Stack

- React 18
- Tailwind CSS
- Recharts
- Lucide Icons
